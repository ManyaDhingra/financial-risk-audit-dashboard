# Financial Risk & Audit Dashboard

## Overview

An interactive Power BI dashboard developed to analyze financial transactions, identify high-risk records, monitor transaction trends, and provide actionable audit insights.

## Features

* Transaction trend analysis by month
* Credit vs Debit comparison
* Territory-wise transaction analysis
* High-risk transaction monitoring
* Risk percentage KPI
* Net balance tracking
* Account-level transaction analysis
* Business insights summary

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Visualization
* Financial Analytics

## Key Metrics

* Total Transactions: 28K
* Total Debit: 104M
* Total Credit: 80M
* Net Balance: 25M
* High Risk Records: 1,396
* Risk Exposure: 5%
